# Task 02 – Prompting for Creativity

## Overview
This task explores how prompt structure, tone, and examples influence creative outputs generated by AI. Using few-shot prompting techniques, different prompt variations are designed to guide the model's creativity, style, and response quality.

## Objective
- Understand creative prompting techniques.
- Use few-shot prompting to guide AI responses.
- Generate creative content using different prompt styles.
- Compare outputs based on creativity, tone, and detail.

## Example Task

### Prompt 1
Generate three innovative startup ideas for college students.

### Prompt 2
You are a successful entrepreneur. Generate three innovative startup ideas for college students. For each idea, include the problem it solves, target audience, and revenue model.

## Sample Output

### Idea 1: StudyBuddy AI
An AI-powered platform that helps students create personalized study plans and practice quizzes.

### Idea 2: Campus Connect
A networking app that connects students with mentors, internships, and project opportunities.

### Idea 3: SkillSwap
A peer-to-peer platform where students can teach and learn skills from each other.

## Process
1. Choose a creative task (startup ideas, stories, advertisements, etc.).
2. Create a basic prompt.
3. Enhance the prompt with examples, tone, and context.
4. Generate multiple outputs.
5. Compare the results for:
   - Creativity
   - Relevance
   - Detail
   - Tone

## Learning Outcomes
- Understanding few-shot prompting.
- Improving AI-generated creative content.
- Learning how examples influence model behavior.
- Developing effective prompt engineering skills.



## Conclusion
This task demonstrates how carefully crafted prompts can significantly improve the creativity, quality, and usefulness of AI-generated content. Adding context, examples, and role-based instructions helps produce more engaging and detailed responses.
